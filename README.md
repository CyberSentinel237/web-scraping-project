# Web Scraping Project 

Project Overview

This repository contains multiple Python web scraping scripts designed to extract real-world data efficiently and accurately. Each scraper is built for a specific use case and includes visual proof of execution and sample outputs.




 Included Scrapers

Image Scraper

Extracts and downloads images from websites

Automatically organizes downloaded content

Includes execution screenshots 




 Quotes Scraper

Scrapes quotes and related metadata from target pages

Outputs structured data for easy use (CSV/text)

Includes screenshots and sample results





Product Scraper (Price Extraction)

Extracts product details including price, name, and links

Handles structured product listings

Includes screenshots and output files demonstrating results



Proof of Work

Each scraper includes:

Screenshots of the script running

Sample output files

Clean and readable results
